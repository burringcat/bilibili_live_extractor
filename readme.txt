# Bilibili Live Stream Extractor
BLSE does one thing and it does it well, that is to extract a playable url from bilibili live room page.
## Usage
### Basic
#### blse <room-ids>
This will print playable urls for each room to STDOUT, one url per line.
example1: blse 1011 1013
example1 output:
https://txy.live-play.acgvideo.com/live-txy/.....
https://txy.live-play.acgvideo.com/live-txy/.....
